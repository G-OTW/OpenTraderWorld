# First steps

You've [installed](/guide/install) OpenTraderWorld and have your admin credentials. Here's how to make it yours.

## Sign in

Open the app and log in at `/login`. If your password was **generated by the installer**, you are asked to **choose a new one on first login** — the generated password only works once.

You can change your username or password any time in **Settings → Account**. Changing your password signs you out of all sessions.

## Set your defaults

Go to **Settings → Defaults** and pick:

- **Language** — applies to the whole app immediately (English, French, German, Spanish, Italian, Portuguese, Chinese).
- **Default currency** and **timezone** — used as the starting values across modules.

## Install your modules

Open **Settings → Modules**. Every module ships with the app; installing one just makes it available in the module switcher and on the dashboard — nothing is downloaded.

- **Install** the modules you want. Start small — you can add more any time.
- Some modules depend on others: **Historical Data Visualization**, **Backtest** and **Quant Tools** all need **Historical Data** (they work on its downloaded datasets).
- **Detach** hides a module and makes it inaccessible; its data is kept unless you also tick *delete data*. You can reinstall any time.

Not sure where to start? See the [module overview](/modules/) for what each one does.

## Get around the app

- **Module switcher** (top left) — jumps between installed modules. Each module owns its whole working area: its own sidebar, pages and content.
- **Dashboard** (home) — a tile-and-widget board of your modules. Click **Edit layout** to add rows, resize tiles, and place widgets (quick-trade form, news feed, portfolio value, todos, timers…). You can create multiple dashboard **pages** and set a default.
- **Settings** — account, defaults, network, modules, data, backup, updates, logs and more. See the [settings reference](/config/settings).

## Recommended next steps

1. **Take a backup habit early** — see [Backup & restore](/guide/backup-restore).
2. If other devices should reach the app, read [Network & remote access](/config/network) before changing anything.
3. Using external market-data providers? Add their API keys in the **Historical Data → Settings** tab as you need them — the app works fine without any.
